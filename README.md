# London Visitor Homepage

This project is a modern, accessible homepage for visitors to London. It highlights top attractions, provides local tips, and introduces a local guide, all with a focus on accessibility and responsive design.

## Features

- Accessible colour palette with high contrast for readability h
- ARIA labels for major sections and headings
- Descriptive alt text for all images
- Responsive layout using Flexbox
- Google Fonts for clean, readable typography
- Insider tips for visiting London
- Section for a local guide

## Technologies Used

- HTML5
- CSS3 (Flexbox, custom properties)
- Google Fonts (Roboto)

## Getting Started

1. Clone this repository:
   ```bash
   git clone https://github.com/tariqkhemili-git/hometown-homepage.git
   ```
2. Open `index.html` in your browser to view the homepage.

## Accessibility

- All images include descriptive `alt` text
- ARIA labels are used for navigation and sectioning
- Colour palette meets WCAG AA contrast standards

## Customisation

- Replace images and text to feature your own city or guide
- Edit tips and activities in `index.html`
